# Suraksha+ Live Tracking Dashboard

This is a React-based web dashboard for live tracking of SOS alerts from the Suraksha+ mobile app.

## Prerequisites

-   Node.js (v18+)
-   AWS Amplify CLI configured
-   Backend deployed (`amplify push` run successfully)

## Setup

1.  Navigate to the `Website` directory:
    ```bash
    cd Website
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Configure AWS Amplify:
    -   The `src/aws-exports.js` file is currently manually configured based on your existing backend.
    -   **Important**: If you re-deploy your backend or change environments, update `src/aws-exports.js` with the new values from `amplifyconfiguration.dart` or `aws-exports.js` generated by Amplify CLI.

## Running Locally

Start the development server:

```bash
npm run dev
```

Open your browser at `http://localhost:5173`.

## Deployment

To build for production:

```bash
npm run build
```

The output will be in the `dist` folder. You can deploy this folder to:
-   AWS Amplify Hosting (`amplify add hosting`)
-   Vercel / Netlify
-   S3 + CloudFront

## Features

-   **Live Dashboard**: Lists all active SOS sessions.
-   **Real-time Map**: Updates user location every 5 seconds.
-   **modern UI**: Glassmorphism design with pulsing indicators.
